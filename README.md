# Sistema de Gerenciamento de Estoque

## Descrição

Sistema web desenvolvido para auxiliar pequenos e médios varejistas no controle de estoque, permitindo cadastrar produtos, registrar entradas e saídas, emitir alertas automáticos de estoque mínimo e gerar relatórios de movimentação.

## Tecnologias

- HTML
- CSS
- JavaScript
- Node.js
- PostgreSQL

## Objetivo

Reduzir desperdícios e facilitar a tomada de decisão através do controle inteligente de estoque.
